Install CUDA and PYTORCH ON A NEW WINDOWS MACHINE (2021)

A Windows Machine with RTX3080 and 11th gen. i-9 CPU

  • Install Chrome
  • Install VSCODE
  • Install Anaconda
  • Install NVIDIA Drive and check it out by “nvidia-smi” which shows the highest CUDA version applicable to this machine
  • Install CUDA globally which requests installation of VStudio and check it out “nvcc –version”
  • Install cuDNN which requires a sign-up at NVIDIA
    • The core thing is copy some files to some directories
  • Install Pytorch which forces version matching
  • Install gensim which require installation of JDK and some JPype as well as JAVA HOME environment variable setting
  • Install KONLPY by some whl file

Everything is smooth hopefully….
